## Installation

1. Clone the project

2. Install all necessary dependencies
    ```python
        pip3 install -r requirements.txt
    ```

3. Migrate database
    ```python
        python3 manage.py migrate
    ```

4. Run Django server
    ```python
        python3 manage.py runserver
    ```
